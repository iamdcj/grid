CSS Grid allows us to set a min and max value for each grid item; min being the narrowest it should render, max being the widest it should render

      
      .grid {
        display: grid;
        max-width: 1216px;
        margin: 0 20px;
        grid-column-gap: 1em;
        grid-template-columns: 100px repeat(2, minmax(240px, 1fr));
      }
